 # Krishna's Birthday Love Challenge

This is a static birthday quiz game from Chloe to Krishna. The surprise unlocks when the player answers at least 7 out of 8 questions correctly.

## Files

- `index.html`: page structure
- `style.css`: romantic UI, animations, and mobile layout
- `script.js`: questions, answers, scoring, photo captions, and reward flow
- `assets/photos/`: compressed photos used by the page
- `assets/music/`: music folder. The page tries to autoplay `Katy Perry-Birthday.mp3` after the first user interaction and falls back to built-in soft music if the file is missing.

## Local Preview

You can open `index.html` directly in a browser.

For the most stable preview, use a local server or VS Code Live Server.

## GitHub Pages

1. Create a GitHub repository.
2. Upload all files in this folder to the repository root.
3. Make sure `index.html` is in the root, not inside an extra folder.
4. Go to `Settings` -> `Pages`.
5. Choose `Deploy from a branch`.
6. Select `main` and `/root`, then save.

## Privacy

If the repository or website is public, anyone with the link may be able to view the photos. Please confirm you are comfortable with that before publishing.
